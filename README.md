# ATENEA

Plataforma web de análisis de riesgo académico, seguimiento estudiantil y visualización de datos educativos.

---

## 🧠 ¿Qué es ATENEA?

ATENEA es un sistema diseñado para identificar riesgo académico en estudiantes mediante el análisis de datos como hábitos de estudio, rendimiento, asistencia y comportamiento académico.

## 📁 Estructura del proyecto

- CSS/ → Estilos del sistema
- JS/ → Lógica del frontend
- HTML/ → Vistas principales
- PHP/ → Backend
- DATABASE/ → Scripts SQL
- IMG/ → Imágenes del sistema
- assets/ → Diagramas y recursos

---

## 🚀 Funcionalidades

- Registro de estudiantes
- Análisis de rendimiento académico
- Detección de riesgo académico
- Dashboard interactivo
- Reportes de datos

---

## 📊 Objetivo

Mejorar la toma de decisiones académicas mediante análisis de datos y detección temprana de riesgo estudiantil.
